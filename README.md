
# iMDb Scraper

iMDb Scraper is a study application that scraped the 'Top 100' movies list in iMDb webpage.


## Running locally

```bash
  git clone https://github.com/mmmartinelli/imdb_scraper.git
```

Go to project's directory

```bash
  cd imdb_scraper
```

Build application

```bash
  cargo build
```

Run application

```bash
  cargo run
```
